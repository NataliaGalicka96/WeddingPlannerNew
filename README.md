## WeddingPlanner
Aplikacja WeddingPlanner oparta na frameworku Symfony.

## Funkcjonalności
* Rejestracja
* Logowanie
* Odliczanie do ślubu
* Lista zadań - możliwość dodawania nowych zadań, odznaczanie zrealizowanych zadań
* Planowanie budżetu
* Sporządzenie  listy gości
* Sporządzenie notatek
* Dodawanie kontaktów
